# 管理后台服务模块

> 知识库管理与运维的可视化后台，提供文档管理、对话记录查看、系统日志等管理能力。

## 模块定位

本模块是面向运维人员和管理员的后端服务，提供知识库全生命周期管理的 REST API 和简单的 Web 管理界面。作为独立可部署的 Spring Boot 应用，通过 RPC 调用核心 AI 服务完成实际业务操作。

## 核心能力

### 知识文档管理

- **文档 CRUD**：知识文档的创建、编辑、查询、删除
- **文档状态控制**：上架/下架/待向量化/向量化完成等状态流转
- **文档启停**：动态控制文档是否参与检索，停用后自动清理向量数据
- **批量导入**：从文件系统批量导入文档，自动匹配知识分类
- **文档向量化刷新**：触发文档重新向量化并更新到向量库
- **文档匹配测试**：测试给定问题与知识库文档的匹配效果

### 知识分类配置

- **动态分类维护**：支持知识领域（scope）和业务领域（business）两级分类的新增、编辑、删除
- **分类启用/禁用**：控制分类是否参与意图匹配
- **排序管理**：自定义分类展示顺序
- **缓存支持**：分类配置带缓存，避免频繁查库

### 对话记录管理

- **对话历史查看**：按对话 ID 查询用户历史问答记录
- **相关度评分**：支持对问答结果进行 0~10 分的相关度反馈
- **Token 用量统计**：按日/按用户维度统计 Token 消耗

### 系统日志

- **操作日志记录**：基于 AOP 切面自动记录接口调用日志
- **日志查询**：支持按时间、类型、方法等条件检索日志

### 消息消费

- **异步消息处理**：监听 MQ 消息队列，处理文档刷新、报告生成等异步任务
- **策略模式**：消息消费采用策略模式，支持多种消息类型扩展

### Web 管理界面

- 内置静态 HTML 页面，无需前端构建工具
- 文档管理、分类配置、Token 统计、批量导入等可视化操作
- 接口调用验证页面

## 架构说明

- 通过 RPC（Dubbo/Feign）调用核心 AI 服务
- 独立数据库配置（MySQL + PGVector 双数据源）
- 集成 Apollo 配置中心（可切换 Nacos）
- 支持 RabbitMQ 异步消息
- 集成 XXL-Job 定时任务调度

## 服务端口

默认端口：`9050`

## 访问地址

- 管理主页：`http://localhost:9050/index.html`
- 文档管理：`http://localhost:9050/knowledge-document.html`
- 分类配置：`http://localhost:9050/knowledge-category.html`
- Token 统计：`http://localhost:9050/token-usage.html`
