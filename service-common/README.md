# 通用服务基础模块

> 全局共享的基础设施层，提供配置管理、中间件集成、工具类、AOP 切面等通用能力。

## 模块定位

本模块是整个框架的"地基"，被所有业务模块共同依赖。集中管理了与具体业务无关的横切关注点：数据库配置、缓存配置、消息队列、线程池、日志记录、异常处理、通用注解等。通过引入本模块，业务模块可以开箱即用地获得完整的基础设施能力。

## 核心能力

### 配置管理

- **Apollo 配置中心集成**：支持从 Apollo 动态拉取配置，运行时热更新
- **MyBatis-Plus 配置**：统一的 ORM 配置（分页插件、自动填充、自定义 SQL 注入器）
- **Web 配置**：CORS 跨域配置、字符编码配置、Servlet 配置
- **REST 客户端配置**：统一的 RestTemplate / RestClient 配置

### 消息队列

- **RabbitMQ 集成**：连接工厂、消息模板、交换机/队列声明
- **阿里云 AMQP 认证**：支持阿里云消息队列的自定义认证提供者
- **消息回调处理**：消息发送确认和消费异常的回调机制

### 缓存

- **Redis 配置**：连接池（Jedis）、序列化方案（Fastjson + String 双序列化器）
- **Redis 操作管理器**：封装常用的 Redis 操作（String、Hash、List、Set、过期等）
- **Spring Cache 集成**：基于 Redis 的 Spring Cache 抽象

### 数据源

- **Druid 连接池**：高性能数据库连接池配置
- **MySQL / PostgreSQL 双数据源**：支持多数据源隔离和独立事务
- **MyBatis-Plus 填充处理器**：自动填充创建时间、更新时间、创建人、更新人

### 线程池

- **MDC 线程池**：自动传递日志 MDC 上下文的线程池，保证异步任务日志可追踪
- **调度线程池**：支持定时任务的 MDC 线程池
- **线程作用域清理**：自动清理线程本地变量，防止线程复用时数据污染

### AOP 切面

- **限流切面**：基于注解的接口限流（支持 IP、用户、全局等维度）
- **请求/响应日志切面**：自动记录接口入参、出参、耗时
- **操作日志切面**：记录用户操作行为日志

### 自定义注解

- **限流注解**：声明式接口限流配置
- **访问计数注解**：接口访问次数统计
- **日志记录注解**：方法级操作日志记录
- **请求/响应日志注解**：方法级入参出参自动记录
- **查询参数注解**：通用查询条件映射
- **提示注解**：自然语言属性描述，用于对象到文本的智能转换

### 异常体系

- **请求异常**：400 参数校验失败
- **业务异常**：通用业务逻辑异常
- **乐观锁异常**：并发更新冲突

### 基础抽象

- **基础 Service**：通用 CRUD 服务抽象（增删改查、分页、批量操作）
- **通用 Mapper**：MyBatis-Plus 增强 Mapper 基类
- **分页参数/结果**：统一的分页请求参数和返回结果封装
- **查询帮助器**：MyBatis-Plus 条件构造辅助工具

### 工具类

- **分页工具**：分页参数转换和结果封装
- **请求上下文**：基于 ThreadLocal 的请求信息持有器
- **安全工具**：用户身份信息获取
- **Spring 容器**：Bean 查找和获取工具
- **转换工具**：对象属性拷贝和类型转换
- **异常工具**：异常堆栈信息提取
- **校验工具**：参数校验辅助方法

### 日志子系统

- **日志实体/DTO**：系统操作日志的数据模型
- **日志 Service**：日志的存储与查询
- **日志 Mapper**：日志表的数据库操作

### 常量定义

- **全局常量**：系统级通用常量
- **业务常量**：业务级通用常量
- **文件常量**：文件处理相关常量
- **MQ 常量**：消息队列名称、交换机等常量
